ℹ️ About This Project

🎯 Project Objective

This system uses supervised machine learning to recommend appropriate drugs and dosages for patients based on their age, gender, and medical condition.

⚙️ System Architecture

Encoding
→
ML Model
→

→

🤖 Models Used

Drug Prediction: Random Forest (200 trees, max_depth=10) — ~94% accuracy

Dosage Prediction: Gradient Boosting (150 estimators) — structured data specialist

📊 Dataset

941 patient records · 5 medical conditions · 15 unique drugs · 5 dosage levels (50–850 mg) · Zero missing values
